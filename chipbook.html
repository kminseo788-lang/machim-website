What You Get

이런 플랫폼이 필요하신가요

콘텐츠를 팔고 싶은데 외부 플랫폼 수수료가 아깝거나,
직접 운영하고 싶은 자체 플랫폼이 필요할 때 맡겨주세요.

01

독립 콘텐츠 판매 플랫폼

전자책·강의·리포트 등 디지털 콘텐츠를 직접 팔 수 있는 자체 플랫폼. 수수료 없이 수익 전액 귀속.

02

회원 + 구독 + 결제 시스템

회원가입부터 결제, 콘텐츠 접근 권한 관리까지 한 세트로. PG사 연동 포함해서 납품합니다.

03

관리자 CMS 포함

코드 몰라도 콘텐츠 등록·수정·발행이 가능한 커스텀 관리자 페이지까지 함께 제공합니다.

Features

구현 가능한 기능들

아래 기능은 모두 이 레퍼런스에서 실제로 구현된 것들입니다.

커스텀 CMS

코드 없이 콘텐츠를 등록·발행하는 5단계 관리자 편집기. 어떤 디지털 콘텐츠 형식에도 적용 가능합니다.

콘텐츠 뷰어

목차·진행률·챕터 이동이 포함된 커스텀 뷰어. 전자책, 리포트, 강의 노트 등 다양한 형식에 적용됩니다.

접근 권한 관리

무료·유료·구독자 전용 콘텐츠를 구분하고 서버 사이드에서 보안 처리. 비인가 접근 차단.

회원 인증 시스템

회원가입·로그인·세션 관리 일체. 소셜 로그인, 이메일 인증 등 필요에 따라 확장 가능합니다.

검색 · 필터 · 태그

키워드 검색 + 다중 태그 필터. 콘텐츠 양이 많아져도 원하는 것을 빠르게 찾을 수 있는 UX.

반응형 · 모바일 최적화

PC·태블릿·모바일 전 디바이스에서 동일한 품질로 동작. 별도 앱 없이 모바일 대응 가능.

Tech Stack

사용하는 기술 스택

빠른 구축과 안정적인 운영을 위한 스택으로 작업합니다.

Frontend Vanilla JS 프레임워크 없이 빠르고 가볍게. 유지보수 용이.
Backend / DB Supabase PostgreSQL 기반. 인증·DB·스토리지·RLS 보안 일체.
배포 Vercel 자동 배포·CDN·커스텀 도메인 연결까지 포함.
결제 PortOne KG이니시스 등 국내 PG사 연동. 결제·환불 처리.

Editor Flow

저작 도구 — 5단계 흐름

누구나 전자책을 쉽게 만들 수 있도록
기본정보부터 발행까지 단계별 UX를 직접 설계·구현했습니다.

1 기본 정보 입력

제목·표지·가격·시리즈 유형을 설정합니다

2 태그 설정

장르·분야·대상 독자·성향 태그로 독자 검색을 돕습니다

3 파트 구성

책의 목차(파트/소제목) 구조를 설계합니다

4 내용 작성

리치 텍스트 에디터로 각 챕터 내용을 작성합니다

5 발행하기

최종 확인 후 발행하면 독자에게 즉시 공개됩니다
