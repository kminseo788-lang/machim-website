📦 진행 절차

착수금 결제 확인 후, 영업일 기준 1~3일 이내 카카오톡으로 먼저 연락드려요. 이후 자료 요청·시안 공유·피드백까지 전 과정은 카카오톡으로 진행됩니다.

✍️ 콘텐츠(글)

사이트에 들어갈 문구는 기획 단계에서 함께 잡아드려요. 다만 정확한 상품 스펙, 법적 문구 등 꼭 그대로 써야 하는 내용은 고객님이 확인·전달해주셔야 해요.

💳 결제 수단 안내

기본 제작비는 카드결제로 진행돼요. 견적 빌더에 없는 추가 요청(11페이지 이상, 특수 기능 등)은 카카오톡으로 별도 협의 후, 그 추가분만 계좌이체로 결제돼요.

카드결제(온라인 결제) 기능이 필요한 사이트라면:
· 결제대행사(PG) 가입, 채널 개설, 이행보증보험 가입, 카드사 심사 신청은 고객님이 직접 진행하셔야 해요 (사업자 명의 계약이라 대행이 불가능해요). 절차는 단계별로 안내해드려요.
· 심사 완료 후 발급되는 채널키·가맹점 코드 두 가지만 저희에게 전달해주시면, 그걸 사이트에 연동해드려요.
· 심사 기간(약 2주) 동안은 계좌이체로 우선 오픈 가능해요.
· 카드사 심사 결과는 고객님의 사업자 상태·서류에 따라 달라져 마침이 결과를 보장할 수 없어요.

☁️ 도메인 · 서버 · 데이터베이스

도메인(사이트 주소), 호스팅(서버), 데이터베이스는 고객님 명의로 직접 가입하셔야 해요. 가입 방법은 저희가 단계별로 안내해드려요.

가입 후 비밀번호는 저희에게 알려주지 않으셔도 돼요 — 대신 "공동 작업자 초대" 또는 "협업 링크" 기능으로 접근 권한만 부여해주시면 돼요 (예: Vercel 팀 초대, Supabase 프로젝트 멤버 추가 등). 이렇게 하면 계정은 항상 고객님 소유로 안전하게 남아있어요.

무료 사용량(트래픽·데이터 용량)을 초과하면 유료 전환이 필요할 수 있어요. 이 경우 사전에 안내드리고, 비용은 고객님 부담이에요.

🖼 폰트 · 이미지

무료 폰트·이미지를 우선으로 사용해요. 유료 소스가 꼭 필요한 경우, 사전 안내 후 실비로 청구돼요.

🔧 오픈 후 추가 요청

오픈 후 3개월 이내의 추가 요청(페이지 추가 등)은 견적 빌더 단가 그대로 적용돼요. 3개월 이후는 별도 협의가 필요해요.

❌ 취소 · 환불

결제 취소·환불은 카카오톡으로 요청해주시면, 이용약관 제7조 기준에 따라 확인 후 결제 시스템에서 직접 처리해드려요. 영업일 기준 3~5일 이내 처리됩니다. 잔금 완납 전까지는 실제 도메인 연결(최종 오픈)이 진행되지 않아요.

🛠 제작 방식

마침은 HTML/CSS/바닐라 JavaScript + Supabase 조합으로 제작해요. 가볍고 빠르며 유지보수가 쉬운 방식이에요. 특정 프레임워크(React 등) 지정이 꼭 필요하신 경우, 사전에 말씀해주시면 가능 여부를 확인해드려요.
